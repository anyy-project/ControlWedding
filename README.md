# Control Wedding

Aplikasi sederhana untuk mengelola undangan pernikahan digital.

## Fitur

- ✅ Informasi Mempelai (Nama, Alamat, Link Maps, Link Undangan)
- ✅ Template Pesan dengan placeholder dinamis
- ✅ Preview pesan sebelum kirim
- ✅ Daftar Tamu
- ✅ Broadcast via WhatsApp

## Cara Pakai

1. Isi informasi mempelai di tab "Informasi"
2. Buat template pesan di tab "Template" (gunakan placeholder: [Nama Tamu], [Nama Mempelai], dll)
3. Tambahkan tamu di tab "Tamu"
4. Kirim undangan ke tamu via WhatsApp

## Placeholder Template

- `[Nama Tamu]` - Nama tamu
- `[Nama Mempelai]` - Nama mempelai
- `[Alamat]` - Alamat acara
- `[Link Maps]` - Link Google Maps
- `[Link Undangan]` - Link undangan digital

## Teknologi

- HTML5
- CSS3
- JavaScript (Vanilla)
- LocalStorage untuk penyimpanan data

## Catatan

- Data disimpan di browser (LocalStorage)
- Untuk broadcast, aplikasi akan membuka WhatsApp Web/App
- Pastikan nomor WhatsApp sudah benar
